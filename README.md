# Property Wizards — Homepage

A single-file, award-led homepage concept for [Property Wizards](https://propertywizards.com.au/),
one of Australia's most awarded buyer's agencies (Perth, WA).

- `index.html` — the complete site (HTML, CSS and JS in one file)
- Brand: their existing logo + blue/gold palette, Poppins
- Focus: awards/recognition as the primary trust driver
- SEO/AEO: semantic markup, meta/Open Graph, `RealEstateAgent` + `FAQPage` structured data

All copy is sourced from Property Wizards' own published material.
Images are hot-linked from the live site and can be swapped for hosted assets.

## Deploy
Static — host `index.html` anywhere (GitHub Pages, Netlify, etc.).
